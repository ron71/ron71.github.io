B) Theme Installation - top

  1. Extract the contents of the ZIP file you have downloaded to somewhere convenient such as your desktop.
  2. Login to your Wordpress administration area.
  3. Select 'Themes' within the 'Appearance' admin menu.
  4. Click the 'Install Themes' tab located at the top of the page.
  5. Select the 'Upload' option to enable the file browser.
  6. Using the file browser, browse to the folder you just created by extracting the ZIP file.
  7. Locate 'wpblink_flatolio.zip' from within the 'Theme' folder and begin uploading.
  8. Once uploaded, you can go ahead and activate the theme.
  9. Configure the theme for use by visiting the 'Theme Options' page under the 'Appearance' menu.

C) Dummy Contents - top

  1. Login to your Wordpress administration area and click 'Import' under the 'Tools' menu.
  2. If this is a fresh installation of Wordpress you'll need to install the importer tool first. Simply click the 'Wordpress' link, followed by the red 'Install Now' button to proceed with the installation, activate the plugin.
  3. Click 'Import' under the 'Tools' menu.
  4. Browse to the XML files we supplied you with (within 'XML' folder).
  5. Follow on-screen prompts and import everything from the file.
  6. Again, if this is a fresh installation of Wordpress, you'll want to remove the default 'Hello World' post also.

D) Theme Options Page - top

  1. Click 'Theme Options' under the 'Appearance' menu.
  2. Visit each sub-section and configure as desired, take note of the accompanying usage text next to each option.
  3. Hit the 'Save Changes' button when done.

E) Featured Images - top

  1. Open or create a post via the 'Posts' menu.
  2. Find the 'Featured Image' block within the sidebar (default position).
  3. Click the 'Set featured image' link to open the file browser window.
  4. Drag & drop your chosen image onto the window to begin uploading (or select a previous image from your library).
  5. Click the blue 'Set featured image' button to confirm your chosen image.
  6. For best results, use images measuring 800*600px or similar.

F) Adding Portfolio posts - top

You can add portfolio posts in the same way as adding normal blog posts except you need to add them via the 'Portfolio' admin menu option.

Add as much text as you need, along with a featured image, gallery images etc..

Portfolio posts are displayed in a different way than normal posts, they show the featured image at the top of the post, and hide the content text behind a toggle button that allows the viewer to read more if they wish to do so.

G) Building a Menu - top

  1. Click 'Menus' under the 'Appearance' menu.
  2. Create a menu and place your chosen links wherever you please.
  3. Hit the 'Save Menu' button to confirm your menu.
  4. Click 'Manage Locations' tab at the top of the screen.
  5. Select your chosen menu from the dropdown list.
  6. Click 'Save Changes' button to confirm your changes.

H) Using Shortcodes - top

  1. Open or create a post via the 'Posts' menu.
  2. Within your post text use any of the following shortcodes.
  3. [red] [blue] [green] [orange] [purple] [leftfloat] [rightfloat]

I) Custom Logo - top

  1. Click 'Logo' under the 'Appearance' menu.
  2. Click 'Upload Logo' button to open the image browser window.
  3. Drag & drop your chosen image onto the window to begin uploading (or select a previous image from your library).
  4. Click 'Use as Logo' button to confirm your image choice.
  5. The logo image will be displayed underneath the sitename & site description text on the homepage only.
